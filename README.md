# Electron To-Do List Desktop App
This is my first Electron app. It is a task list app with saving and file reading functionality.

### Version
1.0

# Usage

### Installation
Install the required dependencies
```javascrip
$ npm install 
```
### Run
To run the app 
```javascript
$ npm start
```
